# e_commerce_Automation
# Playwright Java Automation Framework

## Overview
This project is an end-to-end UI automation framework built using Playwright (Java) and TestNG.

## Features
- Page Object Model (POM)
- Cross-browser testing (Chromium, Firefox, WebKit)
- Retry mechanism for flaky tests
- Screenshot capture on failure
- Business validation (cart total vs item prices)
- Clean selector strategy

## Test Scenario
